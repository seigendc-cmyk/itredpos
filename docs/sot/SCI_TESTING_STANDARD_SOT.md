# SCI Testing Standard SOT

**Owner:** Digital Commerce (Private) Limited  
**Status:** Controlled Source of Truth  

## Required testing

- Unit testing.
- Integration testing.
- Regression testing.
- Acceptance testing.
- Security testing.
- Migration testing.
- Deployment smoke testing.
- Rollback testing.

## Critical workflows

At minimum test:

- Authentication.
- Vendor resolution.
- Licensing.
- Staff access.
- Branch selection.
- Terminal selection.
- Stock receiving.
- Sales.
- Payments.
- Receipts.
- Returns.
- Shift close.
- BI events.
- Audit events.

## Failure rule

A failed critical test blocks release unless the risk is formally documented
and accepted by the release authority.
