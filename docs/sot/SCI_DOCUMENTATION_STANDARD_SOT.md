# SCI Documentation Standard SOT

**Owner:** Digital Commerce (Private) Limited  
**Status:** Controlled Source of Truth  

## Required document information

Every controlled document must state:

- Title.
- Identifier.
- Owner.
- Status.
- Effective date.
- Purpose.
- Scope.
- Responsibilities.
- Decisions.
- Risks.
- Approval status.
- Revision history.

## Currency rule

Outdated documents must be marked:

- SUPERSEDED
- ARCHIVED
- DEPRECATED

An outdated document must not remain presented as the current Source of Truth.
