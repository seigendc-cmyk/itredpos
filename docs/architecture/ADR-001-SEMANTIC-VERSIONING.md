# ADR-001: Adopt Semantic Versioning for SCI

**Status:** ACCEPTED  
**Date:** 13 July 2026  
**Owner:** Digital Commerce (Private) Limited  

## Context

SCI has developed into a multi-product commerce platform requiring controlled,
traceable and recoverable releases.

## Decision

SCI adopts Semantic Versioning.

The first controlled integration baseline is:

SCI v0.8.0-alpha.1

## Consequences

Every formal release must include source references, documentation, testing,
migration status, a Git tag and rollback instructions.
