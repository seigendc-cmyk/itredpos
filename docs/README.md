# SCI Engineering Governance Documentation

This directory contains the controlled engineering governance records for the
seiGEN Commerce Infrastructure.

## Structure

- docs/sot — binding Sources of Truth.
- docs/architecture — architecture decisions.
- docs/roadmap — roadmap and control registers.
- docs/releases — release notes and release register.
- docs/build — baseline and acceptance evidence.
- docs/migrations — migration records.
- docs/rollback — rollback records.
- docs/security — security reviews.
- docs/testing — testing evidence.
- docs/deployment — deployment records.

Where implementation notes conflict with a current approved SOT, the approved
SOT governs until formally amended.
